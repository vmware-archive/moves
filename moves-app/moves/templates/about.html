Scoring-as-a-Service To Operationalize Algorithms For Real-time

This application demonstrates real-time model scoring as a service Pivotal Cloud Foundry (PCF), Pivotal Big Data Suite, Spring Cloud Data Flow, and Python-based open source machine learning. The pipeline applies broadly and would allow us to evaluate and score almost any feed of streaming data - from sensor data to unstructured text data - to drive real-time action.

Take a look at this blog post for more information.

Applications Of Real-time Data Science

While there are many non-real-time data science use cases, real-time data science is a requirement for many applications including the Internet of Things (IoT), fraud detection, risk calculation, health-related alerts, network analysis, marketing personalization, customized rewards programs, and more.

Within the IoT realm, our heavy industry customers strive towards zero unplanned downtime, and we have built predictive models to target their high-risk equipment. For example, the Pivotal Data Science team recently worked with a major oil and gas customer to implement a mud motor failure prediction model. Operationalizing this predictive model has significant impact— according to The American Oil & Gas Reporter, mud motor failure could account for 35% of non-productive time and cost $150,000 per incident. The Pivotal Data Science team has also tackled many other IoT problems—detecting and tracking jet-engine degradation using sensor data, predictive maintenance in automobiles, automobile driver identification, virtual machine capacity forecasting, a connected car pipeline, and more.

How to



  1. Link a smartphone to capture streaming accelerometric data
    • Visit https://dashboard-jyi.cfapps.io/sensor on a smartphone
    • Enter a unique number that will be used to link the sensor to the app
    • Keep the screen on and put the smartphone in your front pocket
    • Note: currently only Safari on iPhone 6s is tested

  2. Build a customized human activity classification model
    • Visit https://dashboard-jyi.cfapps.io/train on a computer
    • Perform three unique activities with the smartphone in your front pocket
    • Click "Build the Model" and proceed to the dashboard

  3. Score the model and view results in the dashboard
    • Visit http://moves.cfapps.pez.pivotal.io/dashboard on a computer
    • Hit start and perform one of the three trained activities

Framework

Model scoring as service

  • Model operationalization: highly scalable using Pivotal Cloud Foundry
    • Application auto-scaling
    • As the data grows, the model scales
  • Application autonomy
    • The model application is independent of other applications = faster development iterations
    • Effective computation resource allocation - using PCF, can dynamically allocate instance count, memory, and CPU resources
  • Multiple applications can access the model scoring service
  • We are leveraging powerful open-source libraries (Python, sklearn, pandas, etc.)

The model scoring workflow. The model scoring PCF application is provided as a service via an API call.

Machine Learning

  • Data: Three dimensional accelerometric time-series data recorded at 15 Hz (15x/second)
  • Training data set: 20 seconds per training activity
  • Feature generation:
    • Two second moving window on training data
    • Time-domain summary statistics and Fourier transform coefficients
  • Algorithm: Random Forest model

Learning Curve for an individual user showing accuracy as a function of activity training time.
